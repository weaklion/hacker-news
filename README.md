# HackerNews API

카테고리 에서 가져온 API를 활용하여 아코디언 패널을 클릭 시 detail정보 출력

## 구조

/api api : 코드가 담겨있는 폴더  
/components : 페이지에서 사용할 컴포넌트들이 담겨있는 폴더  
  /Detail.js : 패널 클릭 시 출력되는 컴포넌트  
  /ExpansionPanel.js : 화면에 표시될 아코디언 패널.   
/App.js : 메인 페이지   
/App.css : 메인 페이지 CSS   

## 사용 라이브러리

Material UI  
https://mui.com/
